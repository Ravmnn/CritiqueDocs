The player (and NPCs if wanted) will be able to craft stuff whenever it has the materials to do so. Depending on the simplicity and kind of object wanted to craft, it could be done directly by hands (like, creating a tab for it in the inventory), for example, in case of band-aids, molotovs, etc. Or via a crafting table if the object in question is too complex, like a gun.

There won't have only a crafting table. The player will have to use the one required depending of the object.

Since this is kind of an action game, the crafting should be fast and practical, so I'd avoid using the approach in games like Minecraft, where the placement of items are manual (Minecraft has an auto placer though).
